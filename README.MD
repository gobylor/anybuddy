# AnyBuddy

Pick any Claude Code /buddy companion you want.

**[anybuddy.cc](https://anybuddy.cc)**

## What is this

AnyBuddy lets you choose exactly which Claude Code buddy you want instead of rolling the dice. Browse all 18 species across every rarity, find the one you like, and get a ready-to-use user ID.

## How it works

Claude Code assigns buddies using wyhash, seeded by your user ID and a global salt. AnyBuddy ships a pre-computed lookup table so you can search by species + rarity and instantly get a matching user ID. Everything runs in your browser.

# AnyBuddy

自由选择你想要的 Claude Code /buddy 伴侣

**[anybuddy.cc](https://anybuddy.cc)**

## 这是什么

AnyBuddy 让你直接选择想要的 Claude Code buddy，不用反复碰运气。浏览全部 18 个物种和所有稀有度，找到心仪的那个，获取可直接使用的 user ID。

## 工作原理

Claude Code 使用 wyhash 算法，以 user ID 和全局 salt 为种子来分配 buddy。AnyBuddy 预计算了查找表，按物种 + 稀有度搜索即可得到匹配的 user ID。一切在浏览器中运行。
